¿Qué es GitHub?

GitHub es una plataforma de alojamiento de código fuente que utiliza el sistema de control de versiones Git. Permite a los desarrolladores colaborar en proyectos de software, gestionar versiones y compartir código de manera eficiente. GitHub también ofrece características adicionales como la gestión de proyectos, documentación, y herramientas para la revisión de código.

Fundadores de GitHub

GitHub fue fundado en 2008 por Tom Preston-Werner, Chris Wanstrath, PJ Hyett y Scott Chacon.
